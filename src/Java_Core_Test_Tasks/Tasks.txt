Работа с стримами и лямбдами:
 - сделать сложный обьект (например машина с разными полями)
 - создать коллекцию (можно разные) из обьектов
 - удалить один/несколько элементов из коллекции через стрим апи
Исключения:
 - вызвать в лямбде метод с checked exception
 - потом с unchecked exception
 - в чем отличие в обработке/использование в лямбде
Многопоточность:
 - реализовать сортировку слияниями через многопоточность
